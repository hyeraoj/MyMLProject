# 🚲 서울시 공공자전거(따릉이) 대여 수요 예측 및 실시간 시뮬레이션 시스템

본 프로젝트는 서울 열린데이터광장의 따릉이 대여 데이터(2023년~2025년)와 기상청의 종관기상관측(ASOS) 서울 지역 일자료 데이터를 결합하여, 기상 변수와 공휴일/주말 정보에 따른 따릉이 대여 수요를 예측하고 분석하는 머신러닝 시스템 및 실시간 대시보드 웹 애플리케이션입니다.

---

## 🌟 주요 기능 (Key Features)

1.  **대용량 데이터 집계 파이프라인:** 
    *   총 5GB가 넘는 3년간의 월별 이용건수 대용량 CSV 파일들을 메모리 과부하 없이 효율적으로 읽어 일자별 합계로 요약하는 청크(Chunk) 데이터 처리 파이프라인 제공.
2.  **머신러닝 예측 모델링:**
    *   기온(평균/최고/최저), 강수량, 풍속, 일사량, 적설량 및 날짜 파생 변수(월, 요일, 주말 여부, 공휴일 여부 등)를 이용한 최적의 수요 예측 모델 학습.
    *   Random Forest Regressor ($R^2 = 0.8887$) 및 Decision Tree, Linear Regression 성능 분석 리포트 제공.
3.  **실시간 웹 시뮬레이션 대시보드 (Streamlit):**
    *   웹 UI 화면에서 마우스 드래그를 통해 실시간으로 [날짜, 기온, 강수량, 풍속, 적설량] 등의 날씨 조건을 변경하며 당일의 대여량을 0.01초 만에 즉각 예측하는 인터랙티브 대시보드.
    *   기온 변화에 따른 예측 수요 곡선 시뮬레이션 및 중요 독립 변수 기여도 그래프 시각화 탭 제공.

## 🛠️ 설치 및 로컬 실행 방법 (Installation & Run Guide)

### 1. 가상환경 세팅 및 필요 라이브러리 설치
터미널을 열고 가상환경을 구성한 뒤, 의존성 라이브러리들을 설치합니다.
```bash
# 가상환경 구성 (이미 구성된 경우 생략 가능)
python -m venv .venv

# 가상환경 활성화
.venv\Scripts\activate

# 패키지 일괄 설치
pip install -r requirements.txt
```

### 2. 머신러닝 학습 파이프라인 구동 (최초 1회)
통합 데이터 로딩 및 EDA 시각화 차트를 빌드하고 최적 모델을 훈련하기 위해 아래 코드를 실행합니다.
```bash
python bite_demand_prediction_1.py
```
*   실행이 완료되면 `output_data/` 폴더 내에 피처 상관관계 히트맵과 최종 피처 가중치 분석 그래프가 이미지로 자동 내보내기 됩니다.

### 3. 실시간 시뮬레이션 웹 대시보드 기동
웹 기반 시뮬레이터를 띄워 브라우저 상에서 마우스 컨트롤로 대여량을 테스트합니다.
```bash
streamlit run app.py
```
*   명령어 입력 시 브라우저가 자동 기동되며, 만약 자동으로 열리지 않을 경우 브라우저 주소창에 `http://localhost:8501`로 접속하십시오.
